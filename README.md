# Simulador FNO (React)

Projeto React com 3 telas (Dados básicos, Seu projeto, Resultado) e barra lateral de progresso.

Instruções:
1. Extraia o conteúdo.
2. `npm install`
3. `npm start`

Observações:
- Tailwind via CDN está incluído no `public/index.html` para facilitar (sem configuração PostCSS).
- `html2pdf.js` é carregado via CDN no `public/index.html` e usado em `App.jsx` através de `window.html2pdf`.
